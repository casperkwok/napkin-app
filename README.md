# Napkin

**估算也要算对。** 给做收入和增长的人用的草稿纸。

→ **[下载 macOS 版](https://github.com/casperkwok/napkin-app/releases/latest)**
→ [napkinmath.app](https://casperkwok.github.io/napkin-app/)

它知道「人」不能加「元」，知道涨了 24% 也可能不显著，
也知道你的假设本来就是个大概。

```
日活 = 12000 users/day
客单价 = 39 ± 15% ¥/user
月收入 = 日活 × 2.4% × 客单价 × 30 day     ¥336,960 ± 50,544

A: 12400 曝光 / 620 转化
B: 12180 曝光 / 683 转化                  +12.15% · p=0.033 ✓ 显著

100 users + ¥50                          量纲不同，不能相加
```

需要 macOS 14 或更新版本 · 已签名公证
